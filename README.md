# drumee-ui-dev-tools
Drumee User Interface Development Helper
```console
npm i --save-dev @drumee/ui-dev-tools
```
Add to your package.json scripts:
    "dev": "drumee-ui-devel"
    "deploy": "drumee-ui-deploy"
Edit files in .dev-tools-rc to fit your environment

```console
npm run dev
```
```console
npm run deploy
```
